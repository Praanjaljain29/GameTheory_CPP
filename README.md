# Game Theory in C++

A learning repository documenting my journey through Game Theory concepts and competitive programming problems using C++.

## Goals

* Understand Game Theory concepts from first principles.
* Implement algorithms and ideas in C++.
* Solve competitive programming problems based on Game Theory.
* Document observations and patterns learned from each problem.

## Topics

* Winning and Losing Positions
* Nim Game
* Misère Nim
* Sprague-Grundy Theorem
* Grundy Numbers
* MEX (Minimum Excluded Value)
* Subtraction Games
* Game Dynamic Programming

## Repository Structure

```text
GameTheory_CPP/
├── README.md
└── Problems/
    └── README.md
```

The `Problems` directory contains Game Theory problems solved during my learning journey.
