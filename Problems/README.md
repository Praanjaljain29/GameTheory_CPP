# Game Theory Problems

This directory contains competitive programming problems based on Game Theory.

Each solution is implemented in C++ and focuses on understanding the core idea behind the problem.

## Problems

| # | Problem  | Concept       | Platform   | Solution       |
| - | -------- | ------------- | ---------- | -------------- |
| 1 | [Nim Game](https://www.hackerrank.com/challenges/nim-game-1/problem) | Nim Sum / XOR | HackerRank | [Solution](./nim_game.cpp) |
| 2 |[Array's Game](https://www.geeksforgeeks.org/problems/arrays-game0327/1) | Parity of Total Moves | GfG | [Solution](./arrays_game.cpp) |

## Learning Approach

For every problem:

1. Understand the game and its rules.
2. Identify winning and losing states.
3. Find the mathematical or algorithmic pattern.
4. Implement the solution in C++.
5. Analyze the time and space complexity.
